What is python3?

python3 is not a label on the screen. It is the name of a program installed on your Mac (like Safari or Calculator). You run it by typing its name in Terminal and pressing Enter.

In this project, the full command is python3 server.py — that means: “run Python, and give it the file server.py.” That file is your small web server for search and quotes.

If Terminal says command not found: python3, install Python from the official site: python.org/downloads (opens in browser), then try again.

Every time you use the app

  1. Start the server — pick one:
    • Mac (easiest): In Finder, go to the folder that contains server.py, then double‑click JohnsStockApp.command. A Terminal window opens; leave it open.
    • From Cursor: Menu Terminal → New Terminal, then see how the terminal works in Cursor/VS Code. Type python3 server.py and press Enter. More detail below.

    If the terminal opens in your home folder, run cd to the directory that contains server.py first — otherwise you start the wrong file or nothing happens.

  2. Read the line it prints — something like John’sStockApp → http://localhost:8844. That number is your port.
  3. Open that exact link in Chrome — use Open app home only after the server is running (same computer). If the port is not 8844, copy the URL from Terminal instead of guessing.
  4. Sanity check (optional): Open API health check — if you see JSON with "ok": true, the Python server is answering; search will work on this origin.

Start / restart / stop the server

Start or restart

  • Double‑click JohnsStockApp.command in your project folder, or
  • In Terminal, cd to your project folder (the one containing server.py), then run: python3 server.py

Stop

Click the Terminal window that is running the server and press Ctrl+C (hold Control, tap C). The site will show “connection refused” until you start it again — that is normal.

Check you are on the right server (not Vite by mistake)

Click: Open API health check. If this fails or you see “unknown route” on Search, the browser tab is probably on the wrong port (another program, not server.py).

Your port is set in .env as PORT=… (default in this project is often 8844). Always match the URL printed when Python starts.

What to put in .env (and what not to paste in chat)

  • Edit the file named .env in the same folder as server.py (in Cursor: click it in the left file list).
  • PORT=8844 (or another free port) — must match what you open in the browser.
  • API keys (Twelve Data, EODHD, Marketstack, optional FMP, …) — one per line, no spaces around =. See .env.example in the project for names.
  • Never paste .env into AI chat or screenshots you share publicly.

If something breaks

  • Connection refused — server not running. Do the 4-step checklist and check health.
  • Search “unknown route” — wrong port / wrong program. Health should work on the same URL as the app.
  • News or corporate shows /api/news / /api/corporate and “Routes: … /api/health” — an old server is still running. Stop it (Ctrl+C), cd to this project, run python3 server.py again; /api/health should list api_revision 10+ and the new routes (incl. /api/fx-eur).
  • 429 / search failed — Yahoo may rate-limit. The server then tries Twelve Data symbol search, then Alpha Vantage SYMBOL_SEARCH (if ALPHAVANTAGE_API_KEY is set). Restart server.py after editing .env. Free tiers are small — use tickers (e.g. NVDA) and pause between tries.
  • Stale page after an update — hard refresh: Chrome Cmd+Shift+R.

Personal use without FMP

If FMP_API_KEY is empty, search uses Yahoo only (no key). The server can cache identical Yahoo searches (see YAHOO_SEARCH_CACHE_SECONDS in .env.example) to reduce rate limits.

APIs (quick)

  • Search — /api/search — FMP if FMP_API_KEY is set; else Yahoo (with country / exchange / currency hints).
  • Quote — /api/quote?symbol=…&exchange=… — Yahoo → Twelve → EODHD → Marketstack → Alpha Vantage → FMP.
  • History — /api/history?symbol=…&range=1y — daily OHLCV (provider order depends on listing).
  • News — /api/news?symbol=…&exchange=… — FMP when FMP_API_KEY is set; otherwise Yahoo Finance RSS (no key).
  • Corporate — /api/corporate?symbol=…&exchange=… — EODHD dividends/splits when EODHD_API_TOKEN is set; FMP dividends, splits, and earnings when FMP is set.
  • FX → EUR — /api/fx-eur — ECB reference rates via Frankfurter (no key); used for Trading 212 EUR totals in the app.
  • Health — /api/health — which keys are configured (no secrets).
